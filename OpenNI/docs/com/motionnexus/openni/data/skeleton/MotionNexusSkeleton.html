Classpublic class MotionNexusSkeleton
InheritanceMotionNexusSkeleton Object
Implements IMotionNexusSkeleton

Contains the data for one skeleton, including overall position, skeleton joint positions, and whether each skeleton joint is tracked successfully. The skeleton position data ranges are in meters.
