Classpublic class MotionNexusDispatcher
InheritanceMotionNexusDispatcher Object

Motion Nexus dispatcher is used to manage the availability of skeletons and cursors of the current users. Enforcing the interface allows for compile time class verification and removes event registration. Instead functions registered by the class with the interface are called directly in a FIFO scenerio. Dispatching is automatically started when an event listener is added, and when all are removed dispatching is stopped. You can call start or stop to managing dispatching for performance control.
